Digital Family Cookbook

Digital Family Cookbook is an open-source recipe management system. It's geared towards families wanting to centralize where they share their recipes. And since you run the system yourself, the data remains yours.

Digital Family Cookbook is written using Node.js, React, & GraphQL.

View on GitHub

About

The idea for this project came about years ago when I realized that I had all my recipes organized, but stored in different places. It made finding them a little challenging sometimes trying to remember which platform I had them stored on. I thought having a centralized app to store them would be nice, and would give me a project to work on. This iteration of the project came about when I decided I wanted to delve into Node.JS and React. I have been doing web development for years, but had yet to use Node or React. This project corrected that.

The recipe management system is built to be self-hosted. This means that all the recipes and all the data belong to you. The platform allows users to input recipes and categorize them. Recipes can also be assigned meats so you can filter recipes you can make with chicken for example.

This platform is geared towards families or groups to share their recipes amongst each other. It's set up to be as private or as public as you choose. Users can be forced to login to view anything, or all visitors can have read access. You can also configure it to allow for anyone to register, or require an invitation code to keep it private. You can read more about how to do all this in the documentation.
